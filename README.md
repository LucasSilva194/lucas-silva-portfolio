# Lucas Silva Portfolio

Modern personal portfolio for Lucas Silva, Full Stack Software Developer in Vila Nova de Gaia, Portugal.

## Stack

- Next.js App Router
- TypeScript
- TailwindCSS
- shadcn-style reusable components
- Framer Motion
- Vercel-ready static deployment

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run lint
npm run build
```

## Editable Content

Most content and links live in `src/data/portfolio.ts`.

Replace these placeholders when ready:

- `profile.links.cv`
- project `demoUrl`
- `metadataBase` in `src/app/layout.tsx`

The CV placeholder points to `/cv/lucas-silva-cv.pdf`. Add the real PDF inside `public/cv/` using that filename or update the link in `src/data/portfolio.ts`.
